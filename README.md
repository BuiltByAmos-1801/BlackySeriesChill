# Blacky Series Chill

Advanced React/Vite artist portfolio for **Sadiq Khan**, the music artist and personal brand behind **Blacky Series Chill**.

## Tech Stack

- React
- Vite
- Tailwind CSS
- Framer Motion
- Lucide React icons
- No PHP

## Run Locally

```bash
npm install
npm run dev
```

Then open the local Vite URL shown in the terminal.

## Build

```bash
npm run build
```

## Sections

- Animated preloader
- Fixed glass navbar
- Cinematic hero with equalizer
- Professional about section
- Spotify artist showcase
- Social hub
- Glassmorphism gallery
- Journey timeline
- Collaboration CTA
- Validated React contact form
- Footer with back-to-top
